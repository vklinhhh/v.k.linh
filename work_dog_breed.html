001 : Dog Breed Prediction

The purpose of this project is to predict dog breeds accurately using various deep learning models and techniques, achieving high accuracy (93%) through a combination of multiple architectures and cross-validation strategies.

  • Programming Language Python
  • Deployment: The model is intended to be deployed on Streamlit
  • Model Training: The code trains multiple deep learning models (InceptionV3, InceptionResNetV2, Xception) and concatenates their extracted features into a single feature vector for classification.
  • Cross-validation: Stratified K-Fold cross-validation with n=3 folds is used to evaluate the model's performance
  • Loss Function and Optimizer: The loss function used is categorical cross-entropy, and the model is optimized using the Adam optimizer

Training Dataset (Source: Kaggle)
