002 : Customer Testimonial Analysis

The report uses R code to set up an environment and perform data analysis on a survey dataset related to customer satisfaction and preferences for various cuisines.
The dataset is imported from a CSV file and cleaned using the "janitor" package.
The analysis includes visualizations using "ggplot2" and "ggpubr" to explore relationships between cuisine types, customer satisfaction, and various factors like approach channels, tools used, price, product quality, fulfillment rate, product variety, shipping service, and customer service.
The data is analyzed based on whether customers will order second time and whether they plan to increase their volume of orders.
The context of this analysis appears to be related to understanding customer preferences and identifying areas for improvement in a food-related business.
